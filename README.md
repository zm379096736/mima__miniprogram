# 秘马群 Dota2 内战小程序

微信原生小程序 MVP，主体围绕“秘马群”内战：

- 今日开 C：展示报名池，系统自动分队。
- 自动分队：每个选手有积分和偏好位置，每队固定 1/2/3/4/5 号位，并尽量压低两队总分差。
- 选手卡片：每个选手都有自己的卡片，展示积分、评级、偏好位置、胜率、鸽子次数。
- 自助资料：选手可在“我的”页自己填写积分、Steam ID、偏好位置。
- 秘马积分榜：以卡片形式展示群内选手。
- 战绩录入：第一版提供天辉胜利的快速录入示例，更新胜场、MVP、压力扣分。

## 打开方式

1. 打开微信开发者工具。
2. 导入项目目录：`mima-dota2-miniprogram`。根目录下应能直接看到 `app.json`、`pages/`、`utils/`。
3. AppID 可先使用测试号或在工具内选择游客模式。

## 自助资料流程

1. 进入“我的”。
2. 修改自评积分。
3. 勾选 1 到 5 号位中的偏好位置，可多选。
4. 保存我的卡片。
5. 回到“开C”页，系统自动分队会使用最新积分和位置偏好。

## 本地测试

当前仓库不依赖第三方包。若本机有 Node.js，可在项目目录运行：

```bash
node --test
```

测试覆盖：

- 选手自助保存积分和偏好位置。
- 分数范围校验。
- 至少选择一个偏好位置。
- 10 人自动分队。
- 天辉/夜魇各自拥有 1 到 5 号位。
- 分队分差控制。
- 胜负、MVP、迟到、爽约、压力扣分的积分更新。

## 后续接云开发

第一版使用 `wx.setStorageSync` 保存本地数据。后续可把这些本地存储替换成云开发集合：

- `players`：选手资料、积分、位置偏好、Steam ID。
- `rooms`：每日内战房间、报名名单、分队结果。
- `matches`：战绩记录。
- `scoreLogs`：积分变动流水。
