# 秘马群 Dota2 内战小程序

微信原生小程序，已升级为微信云开发版本：每个用户通过微信身份生成自己的选手卡，报名、资料、分队、榜单和战绩都走云数据库。

## 打开方式

1. 打开微信开发者工具。
2. 导入项目目录：`mima-dota2-miniprogram`。根目录下应能直接看到 `app.json`、`pages/`、`utils/`、`cloudfunctions/`。
3. 使用你自己的小程序 AppID。

## 云开发配置

1. 在微信开发者工具里点击 **云开发**，开通一个云环境。
2. 复制云环境 ID。
3. 打开 `utils/config.js`，把 `YOUR_CLOUD_ENV_ID` 改成你的云环境 ID。
4. 右键 `cloudfunctions/api`，选择 **上传并部署：云端安装依赖**。
5. 回到模拟器，点击 **编译**。

## 数据集合

云函数会自动创建/写入这些集合数据：

- `players`：选手资料、微信 openid、积分、位置偏好、Steam ID。
- `rooms`：今日房间、报名名单、分队结果。
- `matches`：战绩记录。

第一次进入时，云函数会自动放入 9 个示例选手。当前微信用户进入“我的”页后会自动拥有自己的选手卡。

## 选手流程

1. 进入“我的”。
2. 填选手名、自评积分、Steam ID、偏好位置。
3. 保存我的卡片。
4. 进入“开C”，点击“我要报名”。
5. 满 10 人后点击“系统自动分队”。

## 本地测试

若本机有 Node.js，可在项目目录运行：

```bash
node --test
```

测试覆盖：

- 项目结构和页面 JSON。
- App 启动加载。
- 选手自助保存积分和偏好位置。
- 报名和取消报名。
- 10 人自动分队，每队 1 到 5 号位齐全。
- 胜负、MVP、压力扣分的积分更新。
